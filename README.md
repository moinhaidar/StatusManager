## Setup
- rake db:create
- rake db:migrate
- rake db:seed

## Setup foreman Configuration
web: rails server -p  <port_number>
mail:  mailcatcher


## Access the admin
- Start your server
    ``
        forman start
    ``
- go to http://<host>:<port>/smadmin


## Access to local email interface - Mailcatcher
